# 设置项联动审查（v0.10.0）

本表记录设置项的持久化位置与实际消费者，避免出现“仅有控件、业务未生效”的情况。个性化设置统一经 `preferences:save` 校验后写入本机设置；渲染层启动时重新读取。

| 设置项 | 实际作用目标 | 联动状态 |
| --- | --- | --- |
| 主题 | 根节点 `data-theme`，切换浅色、深色或跟随系统 | 已连接 |
| 强调色 | 根节点 `data-accent`，改变按钮、时间轴与焦点色 | 已连接 |
| 界面缩放 | 根样式 `--ui-scale`，缩放整个工作区 | 已连接 |
| 界面字体 | `--ui-font-size`，应用到导航、标题、按钮、历史记录、设置窗口、详情工具栏与播放器等界面文字 | 已连接；固定像素字号已转换为根相对字号 |
| 玻璃强度 | 根样式 `--glass-alpha`，改变 Liquid Glass 表面透明度 | 已连接 |
| 转写字体 | `--transcript-font-size` 应用于原文；派生的 `--transcript-meta-font-size` 应用于时间、失败切片及原文辅助说明 | 已连接 |
| 智能内容字体 | `--smart-font-size` 应用于智能速览、全文概要、章节/要点/总结和原文内章节卡片；辅助文字按比例派生 | 已连接 |
| AI 对话字体 | `--chat-font-size` 应用于用户/AI 消息、Markdown 和输入框；角色、状态、错误、提示与操作文字按比例派生 | 已连接 |
| 新转写段落长度 | 进入主进程段落聚合器，控制紧凑 2–3 句、标准 3–6 句或长段落 5–9 句 | 已连接；仅应用于新结果，避免覆盖既有手工编辑 |
| 内容密度 | 根节点 `data-density`，调整详情页卡片间距与内边距 | 已连接 |
| 减少动态效果 | 根节点 `data-reduced-motion`，关闭过渡与平滑滚动 | 已连接 |
| 默认倍速 | 新建播放器时初始化 `audio.playbackRate` | 已连接 |
| 默认音量 | 新建播放器时初始化 `audio.volume` 与音量控件 | 已连接 |
| 前进/后退秒数 | 播放器两侧跳转按钮与无障碍标签 | 已连接 |
| 自动跳过静音 | 播放时检查已检测静音区间并跳至区间末端 | 已连接 |
| 最短静音 | 过滤可自动跳过的静音区间 | 已连接 |
| 播放时跟随原文 | 播放位置变化时滚动到对应的近似时间段落 | 已连接 |
| 跳转提前量 | 点击原文近似时间时从估算点之前开始播放 | 已连接 |
| 转写后自动生成智能速览 | 转写完成后调用当前选中的 AI Provider | 已连接 |
| ASR Base URL/入口 | 在按量计费与 Token Plan 入口间切换请求地址 | 已连接 |
| 默认识别语言 | 进入 ASR 请求参数 | 已连接 |
| 自适应并发 | 控制顺序/延迟感知并发执行器；并发模式受 90–92 RPM 滚动节流，429 等待不消耗切片错误预算 | 已连接 |
| AI Provider、Base URL、模型、上下文长度、系统提示词 | 直接组成 OpenAI 兼容对话及智能速览请求；小米内置 Provider 使用 JSON mode，自定义 Provider 自动探测并降级 | 已连接 |

## 明确不在本版本展示的设置

- 说话人区分：当前小米接口返回能力不足以支撑可靠的本地说话人聚类，方案仅记录于 [PENDING_FEATURES.md](./PENDING_FEATURES.md)，未添加占位开关。
- 温度：按已确认需求不向用户开放。

- 自定义播放器已验证播放/暂停状态切换及 1.5x 倍速选择。
